# Prompts Index

保存常用提示词和任务模板。

## 常用模板

- `prd-rewrite-and-complete.md`：PRD 重写、补全、审稿和产品判断。
- `wecom-doc-edit.md`：企业微信文档/PRD 表格的安全编辑。
- `abtest-daily-analysis.md`：ABTest 日报、实验判断和行动清单。
- `frontend-tool-build.md`：前端小工具、页面和可交互原型构建。

建议：

- 一个模板一个 Markdown 文件。
- 文件开头写明用途、输入、输出要求。
- 经常使用的模板放在根目录，实验性模板放 `drafts/`。
