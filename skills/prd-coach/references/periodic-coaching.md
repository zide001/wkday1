# Periodic PRD Coaching

Use this for recurring growth updates.

## Default Cadence

Weekly, unless the user asks for another cadence.

## What To Check

- New personal PRDs added since the last review.
- New learning PRDs added since the last review.
- Whether the user is repeatedly improving or repeating the same blind spot.
- Whether the expression pattern library needs a new pattern.
- Whether the user's clean PRD template should be adjusted.

## Update Format

Keep the update short and actionable:

```markdown
这周 PRD 进步点：
- 

重复出现的问题：
- 

下周只练一个动作：
- 

我已更新的方法论：
- 
```

Do not overwhelm the user with a full audit unless asked.
